Créer un programme qui permet d'ajouter modifier et supprimer dans une liste de produit

  • Référence s'incriment automatiqument mais il peut etre modifier
  • Les champs avec * sont obligatoire
  • La valeur du stock se calcule automatiquement et ne pas pas étre modifier
  • Si nous choisisson la catégorie "Nouriture" une champs "Date d'expiration " s'affiche
  • Le mot de pass est obligatoir pour chaque ajout
  • Le mot de pass ne s'ajoute pas à la list
  • Dans la balise P
    • si l'ajout se pass bien on affiche un message en vert on ajouton la class valid
    • si il ya un probléme on affiche un message en rouge on ajouton la class error
  • Tout les produits s'afficheront dans le tableau en bas
